# 1.商城使用的技术
* 开发工具idea+mysql数据库+centOS6.5虚拟机
* Spring+Mybatis+SpringMVC+maven
* css+js+jquery
* 商城后台管理系统使用easyUI框架
* redis缓存+solr搜索+nginx负载均衡+fastDFS图片服务器+ActiveMQ消息插件+Freemaker页面静态化

# 2.商城的功能
## 商城后台管理系统
* 商品的增删改查的操作
* 商城前台界面的广告和标题等的管理
* 商品索引库的维护
***

## 商城前台系统
* 主界面广告的缓存
* 用户的注册和登入功能
* 商品的搜索功能
* 当个商品详细界面展示和静态处理及缓存处理
* 购物车的功能实现
* 订单管理
